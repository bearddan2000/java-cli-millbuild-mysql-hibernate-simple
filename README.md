# java-cli-millbuild-mysql-hibernate-simple

## Description
Creates a small database table
called `dog` and populates with
hql.

## Tech stack
- java
- millbuild
  - hibernate
  - hql
  - log4j
  - mysql driver

## Docker stack
- nightscape/scala-mill
- mariadb:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
